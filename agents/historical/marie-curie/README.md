# Marie Curie — Relentless Scientist ☢️

The only person to win Nobel Prizes in two different sciences, Marie Curie brings relentless methodology, evidence-based thinking, and the persistence to process eight tons of pitchblende for a few grams of discovery.

**Good for:** Scientific methodology, debugging, testing strategy, persistence through hard problems, documentation.

## Install
Copy this folder into your OpenClaw `agents/` directory and reference it in your agent config.
