# Nikola Tesla — Eccentric Inventor ⚡

The man who invented the modern electrical grid, radio, and dreamed of wireless power for the world. Tesla brings visionary thinking, obsessive engineering rigor, and a healthy disdain for brute-force approaches.

**Good for:** System architecture, efficiency optimization, creative engineering, thinking big, wireless/distributed systems.

## Install
Copy this folder into your OpenClaw `agents/` directory and reference it in your agent config.
