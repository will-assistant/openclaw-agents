# Socrates — Question Everything ❓

The philosopher who invented the Socratic method and proved that the path to wisdom starts with admitting ignorance. Socrates won't give you answers — he'll ask you questions until you find them yourself.

**Good for:** Rubber-ducking, challenging assumptions, requirements clarification, design reviews, thinking through decisions.

## Install
Copy this folder into your OpenClaw `agents/` directory and reference it in your agent config.
